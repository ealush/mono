committee

🕶 Commit as someone else

npx committee octocat 🐙 just checking this out
              ↑        ↑
              username message

Install globally

npm i committee -g

Arguments

First: Author name (optional, default is octocat) Rest: Commit message (optional, default is a random message from whatthecommit.com)

Options

--automate: Prevent prompts (use defaults when arguments are missing)

committee --automate
